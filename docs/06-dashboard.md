# 06. 관리자 대시보드 🖥️

> 브라우저에서 미쿠봇의 서버별 설정과 데이터를 확인할 수 있는 웹 UI입니다.

---

## 대시보드 켜기

`.env`에서 아래 값을 설정합니다.

```env
DASHBOARD_ENABLED=true
DASHBOARD_PORT=3000
```

봇 실행:

```bash
npm start
```

브라우저 접속:

```text
http://localhost:3000
```

---

## 확인 가능한 항목

대시보드에서는 아래 항목을 확인할 수 있습니다.

```text
- 서버별 설정
- AI 채널 설정
- 자동 DJ 상태
- 즐겨찾기 목록
- 플레이리스트 목록
- 최근 별점 기록
- 현재 큐 상태
```

---

## 추천 사용 방식

처음에는 로컬 전용으로 사용하는 것을 추천합니다.

```text
개발 PC에서 실행
→ http://localhost:3000 접속
→ 설정 및 저장 데이터 확인
```

서버에 공개 배포할 경우에는 인증 기능을 추가하는 것이 좋습니다.

---

## 추후 확장 아이디어

```text
- 대시보드에서 자동 DJ on/off
- 대시보드에서 mood 변경
- 플레이리스트 직접 편집
- AI 프롬프트 수정
- 서버별 로그 조회
- 봇 재시작 버튼
- 모델 변경 UI
```
