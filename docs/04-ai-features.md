# 04. AI 기능 🤖

> EXAONE 3.5 기반으로 동작하는 미쿠봇의 AI 기능 정리예요.

---

## 사용 모델

이 프로젝트의 기본 AI 모델은 **EXAONE 3.5 7.8B**입니다.

```bash
ollama pull exaone3.5:7.8b
```

`.env` 설정:

```env
OLLAMA_URL=http://localhost:11434
OLLAMA_MODEL=exaone3.5:7.8b
```

> EXAONE 3.5는 한국어 응답 안정성이 좋아서 미쿠풍 한국어 캐릭터봇에 잘 어울립니다.

---

## AI 멘션 채팅

봇을 멘션하면 미쿠가 답장합니다.

```text
@Miku 오늘 뭐 들을까?
```

예시:

```text
🩵 미쿠
마스터, 오늘은 반짝이는 신나는 곡이 좋을 것 같아요!
무대 조명 켜고 같이 달려볼까요? 🎤✨
```

---

## AI 추천곡

사용자가 원하는 분위기를 입력하면 AI가 추천 검색어를 생성합니다.

```text
/recommend prompt:오늘 좀 우울한데 새벽 감성으로 추천해줘
```

동작 흐름:

```text
사용자 분위기 입력
→ EXAONE 3.5가 추천 검색어 생성
→ YouTube 검색
→ 곡 후보 표시 또는 재생
```

추천 프롬프트 예시:

```text
코딩할 때 들을 신나는 노래
하츠네미쿠 느낌의 보컬로이드
새벽에 듣기 좋은 잔잔한 곡
퇴근길에 들을 애니송
```

## AI DJ 멘트

곡이 시작될 때 미쿠가 짧게 소개 멘트를 해줍니다.

```text
/djment on
```

출력 예시:

```text
🎙️ 미쿠 DJ 멘트
이번 곡은 반짝이는 무대 조명처럼 시작되는 노래예요.
마스터, 오늘도 미쿠랑 같이 리듬 타볼까요? 🩵
```

끄기:

```text
/djment off
```

상태 확인:

```text
/djment status
```

## AI 별점 평가

현재 재생 중인 곡을 AI가 별점과 감상평으로 평가합니다.

```text
/rate
```

감상 힌트 포함:

```text
/rate comment:새벽 감성으로 듣기 좋은지 봐줘
```

출력 예시:

```text
🎧 미쿠의 노래 별점이에요~

별점: ⭐⭐⭐⭐ 4.0 / 5.0

감상평:
제목 분위기 기준으로 잔잔하게 듣기 좋은 곡이에요.

미쿠 코멘트:
마스터, 이 곡은 밤하늘에 작은 조명이 켜지는 느낌이에요 🩵
```

> 실제 음원 분석이 아니라 곡 제목, 아티스트, 사용자 comment 기반의 AI 감상평입니다.

---

## 예능용 음수 별점 모드

특정 문구를 넣으면 예능용으로 음수 별점을 줄 수 있습니다.

```text
/rate comment:음수모드 매운맛으로
/rate comment:억까모드로 평가해줘
/rate comment:독설모드
/rate comment:예능평가
```

출력 예시:

```text
🎭 미쿠의 예능용 매운맛 별점이에요~

예능 별점: 💀💀💀 -3.0 / 5.0

감상평:
제목 분위기만 봐도 예능 심사위원 버튼이 눌린 곡이에요.

미쿠 코멘트:
마스터... 미쿠가 웃으면서 별점을 깎아봤어요 🫠
```

## AI 채널 제한

AI 채팅이 특정 채널에서만 동작하도록 설정할 수 있습니다.

```text
/ai-channel set channel:#미쿠대화방
/ai-channel off
/ai-channel status
```

서버가 커졌을 때 AI가 아무 채널에서나 답장하는 것을 막을 수 있습니다.
